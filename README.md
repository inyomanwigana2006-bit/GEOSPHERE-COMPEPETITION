## 🚀 Cara Pakai

### Lokal (tanpa internet)
1. Download/clone repository ini
2. Pastikan semua file ada sesuai struktur di atas
3. Buka `index.html` di browser

### GitHub Pages (akses online)
1. Push semua file ke repository GitHub
2. Masuk ke **Settings → Pages**
3. Pilih branch `main`, folder `/root`
4. Klik **Save** — website akan aktif di `https://username.github.io/nama-repo`

---
